# 已修复

- 底部状态栏已改为结构化信息区，分别显示状态、当前文件路径、画布统计和最近操作结果。
- 状态栏已支持正常、成功、警告、错误的轻量颜色区分，刷新失败会提示保留最后一次成功视图。
- 文件监听失败不再静默降级，会在状态栏显示 `Watcher unavailable`。
- 视图工具栏已将 `Fit View` 和 `Reset 1:1` 合并为一个视图模式按钮，并优先展示视图、缩放和导出操作。
- 空状态已提供 `Open Canvas` 主按钮和 `Open Recent` 次按钮，并复用现有打开回调。
- Inspector 已改为摘要、Content、Source、Geometry 和 Actions 分区，Source 支持单独复制。
- Geometry 已拆分为紧凑的 `x/y/w/h` 2x2 信息格。
- Canvas 普通边线已弱化，选中节点相关边和关联节点已强化层级。
- 节点已增加轻量类型 badge，group 节点视觉权重已弱化。
- 已为 Inspector 未来编辑控件预留 Actions 区布局空间。
- 顶部栏已改为左侧锚点布局，品牌图标和 `RipCanvas` 靠左聚合，文件菜单紧随其后。
- 顶部栏已移除节点/边统计，避免和底部状态栏重复。
- 底部状态栏已改为文件名优先显示，完整路径作为次级信息截断展示。
- Inspector Actions 区已移除空占位，改为明确 disabled 的未来动作按钮。
- group 节点已进一步弱化为容器视觉，降低阴影、透明度和标题权重。
- 空状态已从画布左上移动到视口中心区域。
- 成功状态已支持短暂高亮后自动回到普通 `Watching` / `Idle` 状态。
- 根据最新截图复查后，顶部栏已从居中布局器改为明确左侧锚定，图标、标题和 `File` 菜单固定从窗口左侧起排。
- 底部状态栏已从居中布局器改为左侧状态、文件信息、画布统计和右侧操作反馈的固定区域布局。
- Inspector 未选中节点时已补充当前画布摘要和快捷操作入口，避免只显示一句空提示。
- 空状态卡片已弱化阴影和边框权重，减少对画布工作区的遮挡感。
- 已修正顶部栏、底部状态栏外层容器未撑满窗口导致整组内容仍被居中的问题。
